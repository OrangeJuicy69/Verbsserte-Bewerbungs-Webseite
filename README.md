Bewerbungs-Webseite

Dies ist meine persönliche Bewerbungswebseite, die als interaktives Portfolio und Online-Visitenkarte dient. Die Seite ist responsiv, modern gestaltet und enthält alle wichtigen Informationen zu meiner Person, meinen Fähigkeiten und Projekten.

Vorschau

Füge hier optional einen Screenshot deiner Webseite ein.

Features

Sticky Navbar für einfache Navigation

Dark/Light Mode per Theme-Switcher

Interaktive Elemente: Hover-Effekte, animierte Buttons, Social-Icons

Portfolio-Sektionen für Projekte, Skills und Kontaktinformationen

Responsives Design für Desktop, Tablet und Mobile

3D-Animationen und Glow-Effekte für Highlight-Elemente

Technologie

HTML5

CSS3 (inkl. CSS-Variablen, Animationen, Flexbox/Grid)

JavaScript für Interaktivität (Theme-Switcher)

Optional: SVG-Icons und Social-Media-Verlinkungen

Installation & Nutzung

Repository klonen:

git clone https://github.com/orange_juicy69/dein-repo.git


In den Projektordner wechseln:

cd dein-repo


Öffne die index.html im Browser deiner Wahl.

Die Webseite benötigt keinen Server und läuft direkt lokal.

Anpassung

Farben und Theme: Anpassbar über CSS-Variablen im :root.

Inhalt: Eigene Informationen, Projekte oder Social-Links können direkt in den HTML-Dateien geändert werden.

Animationen: CSS-Hover- und Glow-Effekte lassen sich einfach erweitern.
